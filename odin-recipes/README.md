# Odin Recipes

This project is part of **The Odin Project** curriculum.  
It is a simple recipe website built using **HTML**.  

## Purpose
The goal of this project is to practice the following skills:
- Setting up a Git repository
- Using Git and GitHub for version control
- Structuring a basic HTML website
- Writing semantic HTML elements
- Linking between multiple pages

## Reflection
By the end of this project, I will have demonstrated my ability to:
- Clone and manage a repository with Git
- Work with `README.md` files
- Build a multi-page HTML site from scratch
- Practice clean coding habits with commits and documentation
